# 2023-5-13 HTML笔记

## 文本格式化

```html

<big>该标签可以放大文字</big>
<small>缩小文字</small>

产生<sub>下标</sub>
产生<sup>上标</sup>

<b>产生粗体文本</b>
```

## 链接

```html
<a href="https://www.runoob.com/" target="_blank">访问菜鸟教程!</a>
为链接添加属性target，设置值为_blank，则链接将在新窗口打开
```





<a href="mailto:xxx@yyy">可以制作电子邮件链接

<body bgcolor="yellow">可以添加背景颜色
